# Clarendon Capital

A real estate investment and financial services website.

**Live site:** [https://clarendon.vercel.app](https://clarendon.vercel.app)

## Pages

- **Home** — `index.html`
- **About Us** — `about.html`
- **Services** — `services.html`
- **Investment Plans** — `plans.html`
- **Contact** — `contact.html`
- **Dashboard** — `dashboard/index.html`
  - Sign In — `dashboard/signin.html`
  - Sign Up — `dashboard/signup.html`

## Tech Stack

- Static HTML5 / CSS3
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Playfair Display + Inter)
- Deployed on Vercel

## Brand

- **Name:** Clarendon Capital
- **Primary colors:** Navy `#0B1F3A`, Gold `#C9A84C`
- **Contact:** info@clarendoncapital.com

## Development

Open any `.html` file directly in a browser, or serve the project with a local static server:

```bash
npx serve .
```
